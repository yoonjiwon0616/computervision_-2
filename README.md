4. 상세 알고리즘 설명 
1) 객체탐지 및 필터링
사용모델 : YOLOv8 (Nano version)
클래스 필터링: 프레임별로 객체를 추출한 뒤, COCO 데이터셋 가중치 프레임워크에서 Class ID 0(Person)에 해당하는 객체만을 필터링함.
신뢰도 임계값 최적화: 야간 노이즈나 고정 사물 오검출을 방지하기 위해 임계값을 0.5로 상향 설정함.

2) 진입 판정 알고리즘
OpenCV의 cv2.pointPolygonTest를 활용한 4-Corner Validation
사람 바운딩 박스의 좌상, 우상, 좌하, 우하 4개 꼭짓점 좌표를 모두 추출한 후, 4개의 모든 꼭짓점이 ROI 내부에 존재할 때(value >= 0)만 진입 상태로 판단함.
3중 상태 분류
완전 외부(0): 다각형 ROI 내부에 포함된 꼭짓점이 0개인 상태
경계선 걸침/접촉(1): 포함된 꼭짓점이 1~3개이거나 꼭짓점과 ROI 외곽선 간의 signed distance 값이 5픽셀 이내인 상태
완전 내부(2): 4개의 꼭짓점이 모두 다각형 ROI 내부에 진입 완료한 상태
: 밖에서 안으로 진입한 객체를 진입으로 판정하고, 안에서 밖으로 나가는 객체를 완료로 판단하여 내부 물체를 객체로 탐지하거나 검출이 중간에 끊기는 오검출을 방지하기 위해 
반드시 외부->접촉->내부인 경우에만 진입, 내부->접촉->외부인 경우에만 완료로 판단함.

3) 이벤트 상태 및 시간 추출
Alarm ON(진입): Is_area_orrupied가 False에서 True로 변한 시점을 기록하고 해당 상태가 1초동안 유지되면 시간을 계산하여 기록.
Alarm OFF(종료): Is_area_orrupied가 False로 변한 순간의 시간을 기록함
